# eslint-config-nuxt

Standard linting configuration for Nuxt.js projects developed by STRG.

## Installation

```bash
# install with npm
npm install -D @strg/eslint-config-nuxt

# or with yarn
yarn add -D @strg/eslint-config-nuxt
```

## Usage

```js
// .eslintrc.js
module.exports = {
  extends: [
    '@strg/eslint-config-nuxt'
  ]
}
```
